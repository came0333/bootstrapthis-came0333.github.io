What is Bootstraps?

Bootstrap is the worlds most popular framework building responsive mobile first sites.

Open Source

Bootstrap is an open source toolkit that develops HTML,CSS and JavaScript. This tool quickly developsa prototype using their Sass variables and mixins, responsive grid system,extensive prebuilt components, and powerful plugins built in jQuery.

Mobile First

Bootstraps includes a powerful mobile first flexbox grid to easily build layouts.Thanks to a twelve column stystem.

First copy/paste the css from https://getbootstrap.com/ to your head tag

Grid

bootstrap grid uses grids,containers,rows and columns to layout and align content. Its resposive.
